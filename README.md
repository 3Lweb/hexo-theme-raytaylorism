# Raytaylorism V1.0a

这是我制作的一款扁平化风格的[Hexo]主题。

[主题预览]

## 安装

```
git clone https://github.com/raytaylorlin/hexo-theme-raytaylorism.git
```

修改 `_config.yml` 中的`theme`一项的值为 `Raytaylorism`即可启用主题。

## 使用的插件

代码语法高亮 [Google-code-prettify]

流量分析 [Google Analytics]
